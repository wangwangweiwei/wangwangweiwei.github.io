                # 机器学习概论

                # 1、什么是机器学习（machine learning）

                概念：让计算机基于数据，构件概率统计模型，用模型对未知的数据进行预测和分析。

                包括两个过程：学习、预测

                运用：文本分类

                特点：

                1. 以计算机或网络为平台

                2. 以数据为研究对象

                3. 对未知数据进行预测（目的）

                学习对象==》数据

                基本假设：同类数据具有一定的统计规律性

                # 2、机器学习分类

                1. 监督学习（数据带标签）

                2. 无监督学习（数据不带标签）

                3. 强化信息

                  补充半监督学习（带标签的数据少，无标签数据多）

                # 3、机器学习方法三要素

                方法 = 模型 + 策略 + 算法

                mine learning

                模型：function set

                策略：loss function

                算法：GD、SGD
